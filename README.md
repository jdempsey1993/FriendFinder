# Friend Finder

Installation
---

The code can be cloned locally to your machine. 

HTTPS:
```
$ git clone https://github.com/jdempsey1993/FriendFinder.git
```
SSH:
```
$ git clone git@github.com:jdempsey1993/FriendFinder.git
```
